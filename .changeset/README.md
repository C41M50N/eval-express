# Changesets

Hello and welcome! This folder has been automatically generated by Changesets to
help you manage release notes and versioning for your package.

## Commands

- `bun run changeset` to create a new changeset
- `bun run version` to consume changesets and update the changelog and version
- `bun run release` to publish changes to npm

If this is your first time using Changesets, see the official documentation:
https://github.com/changesets/changesets
